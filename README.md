# Authentication Backend with NodeJS(NestJS) & Prisma
[Currently deployed on Heroku](https://nestjs-prisma-auth-backend.herokuapp.com/docs)

## ✨ Requirements

- node 14.15.0 또는 16.10.0 이상
- npm 8.7.0

## 📚 Tech Stacks

- Nestjs
- Prisma (ORM)
- Typescript
- Swagger
- Sqlite

## 📦 Init

```shell
$ npm install
```

## 🔨 Run

```shell
# development
$ npm run start:dev

# production
$ npm run build
$ npm run start:prod
```
